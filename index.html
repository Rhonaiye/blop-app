INTRODUCTION TO CSS

WHAT IS CSS?

CSS stands for Cascading Style Sheets.
It is a stylesheet language used to describe the presentation of a document written in HTML or XML.
CSS controls the layout, colors, fonts, and overall visual appearance of web pages.

-->

PURPOSE OF CSS

Allows for consistent styling across multiple pages.

IMPORTANCE OF CSS

Improves user experience with visually appealing web designs.
Enhances accessibility by making content easier to read and navigate.
Boosts website performance by reducing the amount of repeated styling code.

How CSS Works:

CSS works by selecting HTML elements
and applying styles to them. CSS can be applied in three ways: inline, internal,
and external (to be covered later in the week).

The Concept of Cascading: The term "cascading" refers to the way CSS rules are applied in order of priority. The browser determines which style rules apply based on specificity, importance, and source order.

Benefits of Using CSS: Consistency: Ensures a consistent look across all web pages. Efficiency: Allows for reusability of styles, reducing duplication. Maintainability: Changes can be made in one place (CSS file) and affect the entire website.
